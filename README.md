# Sangdon Chatbot API (Netlify)

Netlify Functions backend for the Sangdon Park website chatbot.

## Setup

1. Connect this repository to Netlify.
2. Set environment variable:
   - `GEMINI_API_KEY`
3. Optional Supabase logging variables:
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_KEY` (or `SUPABASE_ANON_KEY`)

## Local Development

```bash
npm install
netlify dev
```

## Main Function

- Endpoint: `/.netlify/functions/chat-ai-driven`
- Source: `netlify/functions/chat-ai-driven.js`

## Knowledge Sync Pipeline

`netlify/functions/knowledge-base.js` is generated by sync script.
Do not hand-edit unless urgently needed.

- Script: `scripts/sync-knowledge-base.js`
- Source pages: `https://sangdon-park.github.io/` (`ko/en/about/courses/publications`)

Run sync (remote site):

```bash
npm run sync:knowledge-base
```

Run sync (local homepage repo):

```bash
npm run sync:knowledge-base:local
```

## Automation

- Workflow: `.github/workflows/sync-knowledge-base.yml`
- Trigger: daily schedule + manual dispatch
- Behavior: sync `knowledge-base.js`, commit, and push only when changed
