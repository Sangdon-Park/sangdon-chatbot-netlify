# Sangdon Chatbot API (Netlify)

Chatbot API for Sangdon Park's website using Netlify Functions and Google Gemini.

## Setup

1. Deploy to Netlify:
   - Connect GitHub repo
   - Set environment variable: `GEMINI_API_KEY`

2. API Endpoint:
   - `https://your-site.netlify.app/.netlify/functions/chat`

## Local Development

```bash
npm install
netlify dev
```
